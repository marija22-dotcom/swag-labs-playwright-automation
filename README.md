# Playwright Automation: SauceDemo Project

This repository contains a professional E2E test automation framework for the SauceDemo application. As a QA engineer focused on precision and ethical software development, I built this framework to demonstrate a scalable and maintainable approach to automation.

## Key Features
- **Page Object Model (POM):** Clean separation between test logic and page-specific actions.
- **Dynamic Data:** Integration with **Faker** to simulate realistic user flows.
- **Resilient Locators:** Strategic use of `data-test` attributes for high test stability.
- **CI/CD Ready:** Automated test execution via **GitHub Actions** on every push to main.
- **Negative Testing:** Includes validation of error messages and edge cases.

## Tech Stack
Playwright | TypeScript | Faker | GitHub Actions

## Quick Start
1. `npm install`
2. `npx playwright install`
3. `npx playwright test`
